Spring Brew Off Master Brewer / Spring Brew Off Club of the Year Scoring Methodology

Each placing entry is given the following points:

  • 1st Place: 0
  • 2nd Place: 0
  • 3rd Place: 0

The following tie-breakers have been applied, in order of priority:

  1. The highest number of first places.
  2. The highest total number of first, second, and third places.
  3. The highest average score.
  4. The lowest number of entries.
