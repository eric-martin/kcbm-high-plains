Best Brewer / Best Club Scoring Methodology

Each placing entry is given the following points:

  • 1st Place: 4
  • 2nd Place: 3
  • 3rd Place: 2
  • HM: 1

The following tie-breakers have been applied, in order of priority:

  1. The highest number of first places.
  2. The highest total number of first, second, and third places.
  3. The lowest number of entries.
  4. The highest total number of first, second, third, fourth (if applicable), and honorable mention places.
  5. The highest average score.
  6. The highest minimum score.
